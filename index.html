O Que São Estruturas de Dados?

Estruturas de Dados de maneira simples são Estruturas destinadas ao armazenamento de dados de forma que após o armazenamento seja possível a manipulação destes dados de maneira eficiente.

Existem Estruturas de Dados tanto estáticas quanto dinâmicas, estruturas estáticas trabalham com um espaço limitado para o armazenamento de dados, enquanto as estruturas dinâmicas como o próprio nome sugere, trabalham com armazenamento de dados dinâmica, não possuindo um limite determinado.

Todas as Estruturas de Dados possuem algumas funções padrão como inserção de um elemento, remoção de um elemento, busca por um elemento e algumas outras funcionalidades destinadas a estruturas específicas.

Em uma pilha só é possível adicionar elementos no topo e retirar elementos do topo.




Em um fila só é possívle adicionar elementos no inicio da fila e retirar elementos do fim da fila.
