O Que São Estruturas de Dados?

Estruturas de Dados de maneira simples são estruturas destinadas ao armazenamento de dados de forma que após o armazenamento seja possível a manipulação destes dados de maneira eficiente.

Existem Estruturas de Dados tanto estáticas quanto dinâmicas, estruturas estáticas trabalham com um espaço limitado para o armazenamento de dados, enquanto as estruturas dinâmicas como o próprio nome sugere, trabalham com armazenamento de dados dinâmico, não possuindo um limite determinado.

Todas as Estruturas de Dados possuem algumas funções padrão como: criação da estrutura, inserção de um elemento e remoção de um elemento. Além disso, cada estrutura possui uma maneira específica de tratar os dados, desde o modo como inserimos os dados, até o modo como podemos ter acesso a eles.

Pilha Estática

Em uma pilha só é possível adicionar elementos no topo e retirar elementos do topo.




Fila Estática

Em uma fila só é possível adicionar elementos no inicio da fila e retirar elementos do fim da fila.




Lista Estática

Em uma lista nós podemos adicionar elementos no começo, no fim e no meio. Também podemos retirar elementos no começo, fim e meio.
