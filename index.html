O Que São Estruturas de Dados?

Estruturas de Dados de maneira simples são Estruturas destinadas ao armazenamento de dados de forma que após o armazenamento seja possível a manipulação destes dados armazenados de maneira eficiente.

Existem Estruturas de Dados tanto estáticas quanto dinâmicas, Estruturas de Dados estáticas trabalham com um espaço limitado para o armazenamento de dados, enquanto as estruturas dinâmicas como o próprio nome sugere, trabalham com armazenamento de dados dinâmica, não possuindo um limite determinado.
